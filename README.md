# Vision Education

An inclusive education platform designed to help visually impaired students learn more effectively using modern technology.

## Features

- Accessible UI
- Voice assistance
- Interactive learning
- Responsive design
- Inclusive educational tools

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```
